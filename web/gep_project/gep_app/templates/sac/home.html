Global Elective Allotment Startup
Follow these steps to start the Global Elective Allotment process

  • 1) Go to root folder of the tathva server. Open 'global.com' folder.
  • 2) Go to /web/gep_project/gep_app/static/gep_app/config folder and change the config page. Set values as follows:
    • ACADEMIC_DATA_PHASE_START_DATE: It is the date and time to open the portal for students and to submit academic details. Format: 'yyyy-mm-ddThh:mm:ss' like '2018-04-07T00:00:00'. Since our server is in chicago, we must convert our time to Chicago time before updating.
    • ACADEMIC_DATA_PHASE_END_DATE: It is the date and time to close the portal to submit academic details. Format: 'yyyy-mm-ddThh:mm:ss' like '2018-04-07T00:00:00'. Since our server is in chicago, we must convert our time to Chicago time before updating.
    • PREFERENCE_SUBMISSION_PHASE_START_DATE: It is the date and time to open the portal to submit the elective preference list. Format: 'yyyy-mm-ddThh:mm:ss' like '2018-04-07T00:00:00'. Since our server is in chicago, we must convert our time to Chicago time before updating.
    • PREFERENCE_SUBMISSION_PHASE_END_DATE: It is the date and time to close the portal to submit the elective preference list. Format: 'yyyy-mm-ddThh:mm:ss' like '2018-04-07T00:00:00'. Since our server is in chicago, we must convert our time to Chicago time before updating.
    • ALLOTMENT_SUBMISSION_PHASE_START_DATE: It is the date and time to publish the elective preference list. Format: 'yyyy-mm-ddThh:mm:ss' like '2018-04-07T00:00:00'. Since our server is in chicago, we must convert our time to Chicago time before updating.
  • 3) Upload Department Data File as csv in the following format. Password of each department will be their username.
    {% csrf_token %}
  • 4) Upload Student Data File as csv in the following format. Username and password of each student will be their username in block letters.
    {% csrf_token %}
  • 4) Upload Faculty Data File as csv in the following format. Username of the faculty will be same as their email and password will be 'FACULTY'.
    {% csrf_token %}
